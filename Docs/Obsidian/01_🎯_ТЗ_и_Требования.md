## 📊 API Требования

### Используемый API: Iconify

**Base URL:** `https://api.iconify.design`

**Endpoints:**

| Endpoint | Метод | Описание |
|----------|-------|----------|
| `/search` | GET | Поиск иконок по запросу |
| `/{prefix}.json` | GET | Детали иконок коллекции |
| `/{prefix}/{icon}.svg` | GET | SVG изображение иконки |

**Пример запроса:**
```
GET https://api.iconify.design/search?query=home
```

**Пример ответа:**
```json
{
  "icons": ["mdi:home", "fa:home"],
  "total": 2,
  "collections": {
    "mdi": { "name": "Material Design Icons" }
  }
}
```

## 🏗️ Архитектурные Требования

### Архитектура: MVP

```
┌─────────────────────────────────────────────────────┐
│                    View Layer                       │
│  ┌─────────────────┐  ┌─────────────────────────┐  │
│  │ ViewController  │  │ TableViewCell           │  │
│  └────────┬────────┘  └───────────┬─────────────┘  │
│           │                       │                │
│           ▼                       ▼                │
│  ┌─────────────────┐  ┌─────────────────────────┐  │
│  │ Presenter       │  │ CellPresenter           │  │
│  └────────┬────────┘  └───────────┬─────────────┘  │
└───────────┼───────────────────────┼────────────────┘
            │                       │
            ▼                       ▼
┌─────────────────────────────────────────────────────┐
│                 Service Layer                       │
│  ┌─────────────────┐  ┌─────────────────────────┐  │
│  │ IconService     │  │ ImageLoader             │  │
│  └────────┬────────┘  └───────────┬─────────────┘  │
│           │                       │                │
│           ▼                       ▼                │
│  ┌─────────────────┐  ┌─────────────────────────┐  │
│  │ NetworkManager  │  │ Cache<URL, UIImage>     │  │
│  └─────────────────┘  └─────────────────────────┘  │
└─────────────────────────────────────────────────────┘
```

### Принципы

| Принцип | Описание | Применение |
|---------|----------|------------|
| **Separation of Concerns** | Разделение ответственности | View ≠ Presenter ≠ Service |
| **Single Responsibility** | Одна ответственность на класс | CellPresenter только для ячейки |
| **Dependency Injection** | Внедрение зависимостей | Assembly создаёт граф |
| **Protocol Oriented** | Программирование на протоколах | Все сервисы через протоколы |

## ⚠️ Типичные Ошибки (из методички)

| Ошибка | Как избежать |
|--------|--------------|
| Нет debouncer | Использовать `CancellableExecutor` |
| Нет пагинации | Добавить `offset`/`limit` в API |
| Нет empty state | 3 состояния: loading, empty, results |
| prepareForReuse | Сбрасывать изображение в ячейке |
| UI не в main thread | Всегда `DispatchQueue.main.async` |
