## 🎯 Назначение CellPresenter

### Что Решает

```
Проблема: Загрузка картинок в UITableViewCell
├── Ячейка уходит в пул при скролле
├── Картинка не успевает загрузиться
├── Ячейка появляется под другим индексом
└── Неправильная картинка отображается!

Решение: CellPresenter для каждой ячейки
- Отменяет загрузку при reuse
- Загружает картинку через ImageLoader
- Уведомляет ячейку через delegate
```

### Зона Ответственности

| Делает | Не Делает |
|--------|-----------|
| ✅ Загрузка изображения | ❌ Не знает о UI элементах |
| ✅ Отмена при reuse | ❌ Не отображает данные |
| ✅ Delegate callback | ❌ Не работает с UILabel |
| ✅ Кэширование (через ImageLoader) | ❌ Не декодирует JSON |

## ⚠️ Проблема Reuse Ячеек

### Схема Проблемы

```
┌─────────────────────────────────────────────────────────────┐
│  1. Пользователь скроллит таблицу вниз                     │
│                                                             │
│  Ячейка 1 (индекс 0) ──┐                                   │
│  │                     │                                   │
│  │ Запрос картинки     │                                   │
│  │ URL: /mdi/home.svg  │                                   │
│  │                     │                                   │
│  │ ────────────────┐   │                                   │
│  │                 │   │                                   │
│  ▼                 │   │                                   │
│  Ячейка уходит     │   │                                   │
│  в пул reuse       │   │                                   │
│                    │   │                                   │
│  2. Пользователь скроллит обратно                          │
│                                                             │
│  Ячейка 1 (индекс 0) ◄─── Получает из пула                │
│  │                     │                                   │
│  │ configure(with:)    │                                   │
│  │ URL: /fa/user.svg   │                                   │
│  │                     │                                   │
│  │ ────────────────┐   │                                   │
│  │                 │   │                                   │
│  │                 ▼   │                                   │
│  │             Картинка из первого запроса!                │
│  │             /mdi/home.svg вместо /fa/user.svg           │
│  │             "Мигание" ячейки                            │
│  └─────────────────────────────────────────────────────────┘
```

### Решение с CellPresenter

```
┌─────────────────────────────────────────────────────────────┐
│  prepareForReuse()                                          │
│                                                             │
│  override func prepareForReuse() {                          │
│      super.prepareForReuse()                                │
│      cellPresenter.cancelLoad()  // ← Отменяем загрузку!    │
│      imageView.image = nil       // ← Сбрасываем image      │
│  }                                                          │
│                                                             │
│  Результат:                                                 │
│  - Старая загрузка отменена                                 │
│  - Ячейка готова к новым данным                             │
│  - Нет "мигания"                                            │
└─────────────────────────────────────────────────────────────┘
```

## 📜 Протокол IconCellPresenterDelegate

### Интерфейс

```swift
import UIKit

/// Протокол делегата для уведомления ячейки о загрузке изображения.
/// 
/// Используется в IconCellPresenter для callback в ячейку.
/// Реализуется в IconTableViewCell.
protocol IconCellPresenterDelegate: AnyObject {
    
    /// Изображение загружено (или взято из кэша).
    /// - Parameters:
    ///   - presenter: Presenter, который загрузил изображение
    ///   - image: Загруженное изображение (nil если ошибка или нет данных)
    func imageLoader(
        _ presenter: IconCellPresenter,
        didLoad image: UIImage?
    )
    
    /// Произошла ошибка при загрузке изображения.
    /// - Parameters:
    ///   - presenter: Presenter
    ///   - error: Ошибка
    func imageLoader(
        _ presenter: IconCellPresenter,
        didFailWith error: Error
    )
}
```

### Реализация в TableViewCell

```swift
extension IconTableViewCell: IconCellPresenterDelegate {
    
    func imageLoader(
        _ presenter: IconCellPresenter,
        didLoad image: UIImage?
    ) {
        // Проверяем, что ячейка всё ещё актуальна
        // (не была переиспользована для других данных)
        imageView.image = image
        
        // Анимация появления
        if image != nil {
            UIView.transition(
                with: imageView,
                duration: 0.2,
                options: .transitionCrossDissolve,
                animations: {}
            )
        }
    }
    
    func imageLoader(
        _ presenter: IconCellPresenter,
        didFailWith error: Error
    ) {
        // Показываем placeholder
        imageView.image = UIImage(named: "icon_placeholder")
        
        // Логирование (опционально)
        print("CellPresenter: Failed to load image: \(error.localizedDescription)")
    }
}
```

## 💻 Реализация IconCellPresenter

### Полная Версия

```swift
import UIKit

/// Presenter для отдельной ячейки таблицы.
/// 
/// Отвечает за:
/// - Загрузку изображения иконки
/// - Отмену загрузки при reuse ячейки
/// - Уведомление делегата о результате
/// 
/// Архитектура: MVP (Cell Presenter layer)
final class IconCellPresenter {
    
    // MARK: - Properties
    
    /// Делегат для уведомления о результате (ячейка)
    private weak var delegate: IconCellPresenterDelegate?
    
    /// Сервис загрузки изображений
    private let imageLoader: ImageLoaderProtocol
    
    /// ID текущей задачи загрузки
    /// -1 означает, что активной задачи нет
    private var currentTaskId: Int = -1
    
    // MARK: - Initialization
    
    /// Инициализация с зависимостями.
    /// - Parameters:
    ///   - delegate: Делегат для уведомления (ячейка, weak)
    ///   - imageLoader: Сервис загрузки изображений
    init(
        delegate: IconCellPresenterDelegate,
        imageLoader: ImageLoaderProtocol
    ) {
        self.delegate = delegate
        self.imageLoader = imageLoader
    }
    
    // MARK: - Public Methods
    
    /// Загрузить изображение для иконки.
    /// - Parameter url: URL изображения (может быть nil)
    func loadImage(from url: URL?) {
        // 1. Если URL nil — показываем пустое изображение
        guard let url = url else {
            delegate?.imageLoader(self, didLoad: nil)
            return
        }
        
        // 2. Отменяем предыдущую загрузку
        cancelLoad()
        
        // 3. Запускаем новую загрузку
        currentTaskId = imageLoader.loadImage(from: url) { [weak self] image in
            guard let self = self else { return }
            
            // 4. Уведомляем делегат
            self.delegate?.imageLoader(self, didLoad: image)
        }
    }
    
    /// Отменить текущую загрузку.
    /// Вызывается при reuse ячейки.
    func cancelLoad() {
        // Отменяем загрузку в ImageLoader
        imageLoader.cancelLoad(taskId: currentTaskId)
        currentTaskId = -1
    }
}
```

## 🔗 Интеграция с ImageLoader

### Схема Работы

```
┌─────────────────────────────────────────────────────────────┐
│  1. Ячейка запрашивает загрузку                            │
│                                                             │
│  IconTableViewCell.configure()                              │
│         │                                                   │
│         ▼                                                   │
│  cellPresenter.loadImage(from: url)                         │
│                                                             │
└─────────────────────────────────────────────────────────────┘
                            │
                            ▼
┌─────────────────────────────────────────────────────────────┐
│  2. CellPresenter отменяет старую загрузку                 │
│                                                             │
│  IconCellPresenter.loadImage()                              │
│         │                                                   │
│         ▼                                                   │
│  cancelLoad()  ← Отмена предыдущего taskId                  │
│                                                             │
└─────────────────────────────────────────────────────────────┘
                            │
                            ▼
┌─────────────────────────────────────────────────────────────┐
│  3. CellPresenter запускает новую загрузку                 │
│                                                             │
│  imageLoader.loadImage(from: url) { image in               │
│      // Загрузка из кэша или сети                          │
│  }                                                          │
│                                                             │
│  Возвращает taskId для возможной отмены                    │
│                                                             │
└─────────────────────────────────────────────────────────────┘
                            │
                            ▼
┌─────────────────────────────────────────────────────────────┐
│  4. ImageLoader проверяет кэш                              │
│                                                             │
│  if let cached = cache.value(forKey: url) {                │
│      completion(cached)  ← Быстро!                         │
│  } else {                                                    │
│      // Загрузка из сети                                   │
│  }                                                          │
│                                                             │
└─────────────────────────────────────────────────────────────┘
                            │
                            ▼
┌─────────────────────────────────────────────────────────────┐
│  5. Callback в CellPresenter                               │
│                                                             │
│  imageLoader.loadImage { [weak self] image in              │
│      self?.delegate?.imageLoader(self, didLoad: image)     │
│  }                                                          │
│                                                             │
└─────────────────────────────────────────────────────────────┘
                            │
                            ▼
┌─────────────────────────────────────────────────────────────┐
│  6. Делегат обновляет UI                                   │
│                                                             │
│  IconTableViewCell.imageLoader(didLoad: image)              │
│         │                                                   │
│         ▼                                                   │
│  imageView.image = image                                    │
│                                                             │
└─────────────────────────────────────────────────────────────┘
```

## ✅ Best Practices

### 1. Weak Delegate

```swift
// ✅ Хорошо
final class IconCellPresenter {
    private weak var delegate: IconCellPresenterDelegate?
}

// ❌ Плохо
final class IconCellPresenter {
    private var delegate: IconCellPresenterDelegate  // Retain cycle!
}
```

**Почему:** Ячейка → CellPresenter (сильная), CellPresenter → Delegate (слабая).

---

### 2. Отмена в prepareForReuse

```swift
// ✅ Хорошо
override func prepareForReuse() {
    super.prepareForReuse()
    cellPresenter?.cancelLoad()  // Отмена загрузки
    imageView.image = nil        // Сброс изображения
}

// ❌ Плохо
override func prepareForReuse() {
    super.prepareForReuse()
    // Загрузка не отменяется!
    // Картинка может загрузиться для старых данных
}
```

---

### 3. Проверка на Актуальность

```swift
// ✅ Хорошо: проверяем в delegate
func imageLoader(_ presenter: IconCellPresenter, didLoad image: UIImage?) {
    // Ячейка могла уйти в пул и вернуться с другими данными
    // Проверяем, что данные актуальны
    imageView.image = image
}

// ❌ Плохо: слепое обновление
func imageLoader(_ presenter: IconCellPresenter, didLoad image: UIImage?) {
    // Может обновить чужую ячейку!
    imageView.image = image
}
```

---

### 4. Обработка nil URL

```swift
// ✅ Хорошо
func loadImage(from url: URL?) {
    guard let url = url else {
        delegate?.imageLoader(self, didLoad: nil)
        return
    }
    // Загрузка
}

// ❌ Плохо
func loadImage(from url: URL?) {
    // url может быть nil!
    imageLoader.loadImage(from: url!)  // Crash!
}
```

## ⚠️ Антипаттерны

### 1. Загрузка в Ячейке Напрямую

```swift
// ❌ Плохо: ячейка знает про URLSession
class IconTableViewCell: UITableViewCell {
    
    func configure(url: URL) {
        URLSession.shared.dataTask(with: url) { data, _, _ in
            let image = UIImage(data: data!)
            DispatchQueue.main.async {
                self.imageView.image = image
            }
        }.resume()
    }
}

// ✅ Хорошо: через Presenter
class IconTableViewCell: UITableViewCell {
    
    func configure(viewModel: IconViewModel, presenter: IconCellPresenter) {
        presenter.loadImage(from: viewModel.iconImageURL)
    }
}
```

---

### 2. Нет Отмены Загрузки

```swift
// ❌ Плохо
final class IconCellPresenter {
    
    func loadImage(from url: URL) {
        imageLoader.loadImage(from: url) { ... }
        // Нет сохранения taskId!
        // Нет метода cancelLoad()
    }
}

// ✅ Хорошо
final class IconCellPresenter {
    
    private var currentTaskId: Int = -1
    
    func loadImage(from url: URL?) {
        cancelLoad()  // Отмена старой
        currentTaskId = imageLoader.loadImage(from: url) { ... }
    }
    
    func cancelLoad() {
        imageLoader.cancelLoad(taskId: currentTaskId)
    }
}
```

---

### 3. Сильная Ссылка на Делегата

```swift
// ❌ Плохо
final class IconCellPresenter {
    
    private var delegate: IconCellPresenterDelegate
    
    init(delegate: IconCellPresenterDelegate) {
        self.delegate = delegate  // Retain cycle!
    }
}

// ✅ Хорошо
final class IconCellPresenter {
    
    private weak var delegate: IconCellPresenterDelegate?
    
    init(delegate: IconCellPresenterDelegate) {
        self.delegate = delegate
    }
}
```

---

### 4. UI Не в Main Thread

```swift
// ❌ Плохо
func imageLoader(_ presenter: IconCellPresenter, didLoad image: UIImage?) {
    // Может быть в background thread!
    imageView.image = image  // Crash!
}

// ✅ Хорошо
func imageLoader(_ presenter: IconCellPresenter, didLoad image: UIImage?) {
    // ImageLoader уже возвращает в main (благодаря DispatchQueue.main.async)
    imageView.image = image
}
```

## 🧪 Тестирование

### Mock Delegate

```swift
import UIKit

/// Mock Delegate для тестов CellPresenter.
final class MockCellPresenterDelegate: IconCellPresenterDelegate {
    
    var imageDidLoadCalled = false
    var imageLoadFailedCalled = false
    
    var lastImage: UIImage?
    var lastError: Error?
    
    func imageLoader(
        _ presenter: IconCellPresenter,
        didLoad image: UIImage?
    ) {
        imageDidLoadCalled = true
        lastImage = image
    }
    
    func imageLoader(
        _ presenter: IconCellPresenter,
        didFailWith error: Error
    ) {
        imageLoadFailedCalled = true
        lastError = error
    }
}
```

### Unit Тест Пример

```swift
import XCTest

final class IconCellPresenterTests: XCTestCase {
    
    var presenter: IconCellPresenter!
    var mockDelegate: MockCellPresenterDelegate!
    var mockImageLoader: MockImageLoader!
    
    override func setUp() {
        super.setUp()
        mockDelegate = MockCellPresenterDelegate()
        mockImageLoader = MockImageLoader()
        presenter = IconCellPresenter(
            delegate: mockDelegate,
            imageLoader: mockImageLoader
        )
    }
    
    func testLoadImage_NilURL() {
        // When
        presenter.loadImage(from: nil)
        
        // Then
        XCTAssertTrue(mockDelegate.imageDidLoadCalled)
        XCTAssertNil(mockDelegate.lastImage)
    }
    
    func testLoadImage_ValidURL() {
        // Given
        let testURL = URL(string: "https://example.com/icon.png")!
        mockImageLoader.mockImage = UIImage()
        
        // When
        presenter.loadImage(from: testURL)
        
        // Then
        XCTAssertTrue(mockImageLoader.loadImageCalled)
        XCTAssertEqual(mockImageLoader.lastURL, testURL)
    }
    
    func testCancelLoad() {
        // Given
        let testURL = URL(string: "https://example.com/icon.png")!
        presenter.loadImage(from: testURL)
        
        // When
        presenter.cancelLoad()
        
        // Then
        XCTAssertTrue(mockImageLoader.cancelLoadCalled)
    }
}
```
