# 📘 Как Использовать Базу Знаний

## 🎯 Назначение

Эта папка содержит полную документацию по проекту IconSearcher в формате **Obsidian** — системы связанных заметок для обучения и справки.

---

## 📥 Установка Obsidian

1. **Скачай** с официального сайта: [obsidian.md](https://obsidian.md)
2. **Установи** на свой компьютер (macOS, Windows, Linux)
3. **Открой** папку: `/Volumes/AI_DATA/develop/projects/IconSearcher/Docs/Obsidian/`

---

## 🗺️ Навигация

### Начни с Индекса

Открой файл **`00_📚_IconSearcher_Индекс.md`** — это главная страница со ссылками на все заметки.

### Используй Ссылки

Внутри заметок есть ссылки вида `[[Название Заметки]]`:
- **Кликни** на ссылку → перейдёшь к соответствующей заметке
- **Cmd/Ctrl + Клик** → откроешь в новой вкладке

### Граф Связей

В Obsidian есть режим **Graph View** (⌘G на Mac):
- Показывает связи между заметками
- Помогает увидеть общую картину

## 📝 Как Обновлять

### Добавляй Заметки

1. Создай файл с префиксом (например, `10_Новая_Тема.md`)
2. Добавь metadata сверху:
```markdown
---
tags: [тег1, тег2]
parent: [[00_📚_IconSearcher_Индекс]]
related: [[Связанная_Заметка]]
---
```
3. Добавь ссылку в `00_📚_IconSearcher_Индекс.md`

### Обновляй Существующие

1. Открой заметку
2. Добавь новую информацию
3. Обнови дату в конце: `*Последнее обновление: 2026-02-27*`

## 🎯 Сценарии Использования

### Сценарий 1: Перед Написанием Кода

1. Открой заметку по теме (например, `04.1_Cache_Кэширование.md`)
2. Прочитай раздел "Реализация"
3. Скопируй код
4. Вставь в свой проект

## 💡 Советы

### 1. Используй Backlinks

В Obsidian есть панель **Backlinks** справа:
- Показывает, какие заметки ссылаются на текущую
- Помогает найти связанные темы

### 2. Создавай Свою Карту Знаний

Не бойся добавлять:
- Свои заметки
- Свои примеры
- Свои выводы

### 3. Экспортируй в PDF

Для чтения без Obsidian:
- Файл → Экспорт → PDF
- Или используй Obsidian Publish

### 4. Синхронизируй

Для доступа с разных устройств:
- Obsidian Sync (платно)
- iCloud Drive (бесплатно для macOS/iOS)
- Git (бесплатно, для технических)
